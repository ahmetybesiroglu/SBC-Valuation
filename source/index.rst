Welcome to SBC-Valuation's Documentation
========================================

This documentation provides an overview of the SBC-Valuation project, which automates the valuation of stock-based compensation (SBC) options using the Black-Scholes model. It integrates with Airtable, retrieves historical data from Yahoo Finance, and calculates essential financial metrics like volatility and treasury yields.
